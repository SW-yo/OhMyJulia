### 1. 乱数生成
- `rand` : 乱数生成
- `rand()` の引数 : `rand(vec)` は内部で `vec[rand(1:length(vec))]` と同等の処理を行っており、非常に効率的に実装されている。
- 基本的な使い方
```julia
vec = [10, 20, 30, 40, 50]

# Vectorからランダムに1つ取得
item = rand(vec)
```
- 要素ではなくインデックスを取る
```julia
# vecのランダムなインデックスを取る
idx = rand(eachindex(vec))
item = vec[idx]
```
- 複数を取る
```julia
items = rand(vec, 3) # 3つ抽出
```
- 重複なしで取る
```julia
# StatsBaseを使う
using StatsBase
items = sample(vec, 3, replace=false)
```
- ランダムシードを固定する : 乱数生成器を作成しない（`Random.seed!()`を使う）とグローバルに固定される。
```julia
# MersenneTwister使用時は必要（標準のrand()では不要）
using Random

# 独立した乱数生成器を作成
rng = MersenneTwister(123)

# rand(生成器, ...) の形式で使う
val = rand(rng, 1:100)
```
