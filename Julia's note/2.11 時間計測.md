### 1. @time
- 関数の先頭に`@time`を付ける
例：
```
A = rand(1000, 1000)
B = rand(1000, 1000)
@time C = A * B
```

### 2. @benchmark
- ベンチマークをする場合は`BenchmarkTools`パッケージをインストール（pkgモード、初回のみ）
```
add BenchmarkTools
```
- 東大テキストの方法はpkgモードに入っていない場合
```
import Pkg
Pkg.add("BenchmarkTools")
```
- 計測時は`Benchmark`をインポート
```
using BenchmarkTools
```
