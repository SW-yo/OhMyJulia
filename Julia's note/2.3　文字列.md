### 1. 文字列
- ダブルコーテーションで囲む。
- 表示は`println()`
- `println(文字列, 数字)`で文字列と数字を組み合わせて返してくれる。
- 連結は`"文字列"*"文字列"`で連結。
- 文字列は配列扱い（スライス可能）

### 2. マルチバイトの罠
- 例文「`sentence="次回は線形代数（大学受験？）の問題について、チームで取り組んでもらいます。現在線形代数の講義で使っている教科書とか、あるいは発展的なトピックでもよいので関連しそうな数学の本などを持ってきてください。"`」から「線形代数」を抜き出す方法。
- 方法A：`nextind` を使って「○文字目」を指定す
	「○番目の文字」の位置を正確に特定する方法。```
```
# 4文字目から7文字目まで
s_start = nextind(sentence, 0, 4) # 4文字目の開始バイト位置
s_end   = nextind(sentence, 0, 7) # 7文字目の開始バイト位置```

println(sentence[s_start:s_end]) # これで「線形代数」が出る
```
- 方法B：`collect` で一旦配列にする（直感的）
	一旦、1文字ずつの配列にバラしてしまえば、`5:8` で直感的にスライスできる。
```
chars = collect(sentence)
println(join(chars[4:7])) # 配列を結合して表示
```
- 方法C：`findfirst` を使う（一番おすすめ）
	文字数を数える必要すらなくなる。
```
range = findfirst("線形代数", sentence)
println(sentence[range])
```
- LLMが文字を処理するとき、「1文字が何バイトか」、「どう区切るか（**トークナイズ**）」という問題は非常に重要になる。
