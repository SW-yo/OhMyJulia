### 1. if文
- `if`
例：
```
a = 8
if a > 5
	println("そのとおり")
end
```
- if文の中はインデント下げ
- if文の終了は明示的に`end`

### 2. else
- `else`
例：
```
s = "fuga" 
if s == "hoge"
	println("Hoge!")
else
	println("Not Hoge!")
end
```

### 3. 条件追加
- `elseif`
例：
```
x = 0.5
if x < 0.3
	println("xは小さい")
elseif x < 0.7
	println("xは中くらい")
else
	println("xは大きい")
end
```
- Juliaに`match-case`や`switch=case`はない。主に`if-elseif-else`文やポリモーフィズム（多重ディスパッチ）を利用して条件分岐を行う設計思想であるため、専用の`switch`キーワードは優先的に実装されていない。

### 4. ポリモーフィズム（多態性・多様性）とは
- 同じ名前の命令（メソッドや関数）を呼び出しても、受け取るモノ（型やオブジェクト）によって、実際の動きが自動的に切り替わる仕組み
- 「中身を気にせず命令できる」、「コードがスッキリする」、「拡張が楽」といったメリットがある。
- Juliaでは特に**多重ディスパッチ**（関数に渡された「すべての引数」の型の組み合わせを見て、実行するメソッドを自動で切り替える仕組み）を言語の核に据えている。
- Pythonなどの動的言語では、実行するたびに「この変数の型は何かな？」と都度確認（**ランタイム・チェック**）するため、そのオーバーヘッドが積み重なって遅くなる。
- Juliaはこの問題を「**型推論**」と「**JIT（Just-In-Time）コンパイル**」の組み合わせで解決している。
- **型推論**：「この型なら、次に実行される計算はこの型になるはずだ」と、コードの先まで型を予測
- **JITコンパイル**：「この型の組み合わせなら、CPUが一番速く動かせる機械語の命令がこれだ」というセットをプログラム実行の瞬間に作り、それをキャッシュ

### 5. while文
- `while`
例：
```
i = 1
while i <= 5
	println(i)
	i += 1
end
```

### 6. for文
- `for`
例：
```
for i in 1:5
	println(i)
end
```
```
x = [10; 20; 30; 40; 50]
for v in x
	println(v)
end
```
※Cっぽく書けるが上の書き方の方が多いらしい。
```
x = [10; 20; 30; 40; 50]
for i in 1:5
	println(x[i])
end
```
- 二重ループ
例：
```
for i in 1:2
	for j in 1:3
		println("i=", i, ", j=", j)
	end
end
```
```
for i in 1:2, j in 1:3
	println("i=", i, ", j=", j)
end
```
- 行列表示
例：
```
# 行列表示

A = [1 2 3;
4 5 6;
7 8 9]

# これだと列→行の順で要素の抜き出しになる。
# Juliaは列優先で処理をする。
for v in A
    println(v)
end

# この書き方は縦に1列の要素の抜き出しになる。
for  i in 1:1
    for v in A[i:i, 1:3]
        println(v)
        println(v')
    end
end

# 地道にやる方法
for i in 1:size(A, 1)        # 行の数だけループ
    row = A[i, :]            # i行目の全列を取り出す
    println(join(row, "  ")) # 要素をスペースでつないで1行で表示
end

# 地道を1行ループで書く方法
# 行i と 列j を同時に回す
for i in 1:3, j in 1:3
    print(A[i, j], " ")
    if j == 3
        println() # 3列目まで行ったら改行
    end
end

#  eachrow(A) を使うと、v に「1行分」が順番に入る。
for v in eachrow(A)
    println(v)
end

# v には [1, 4, 7], [2, 5, 8], [3, 6, 9] が順番に入る。
for v in eachcol(A)
	println(v)
end

# 出力だけならこれが一番ちゃんしている。
display(A)
```
