### 1. ディレクトリの移動・確認

| コマンド | 説明 |
| :--- | :--- |
| `pwd()` | カレントディレクトリのパスを取得 |
| `cd("path")` | 作業ディレクトリを移動 |
| `readdir()` | ファイル・フォルダ名の一覧を配列で取得 |
| `homedir()` | ホームディレクトリのパスを取得 |
- Windowsのパスを使う場合は文字列の前に`raw`を付けて`cd(raw"path")`のように指定。
- もしくは`\`を`/`に置き換えるか`\\`のように2つ重ねる（エスケープ）。

### 2. ファイル・フォルダ操作

| コマンド | 説明 |
| :--- | :--- |
| `mkdir("dir")` | ディレクトリを作成 |
| `mkpath("a/b/c")` | 中間ディレクトリも含めて作成 |
| `rm("path")` | ファイルまたは空ディレクトリを削除 |
| `rm("path", recursive=true)` | ディレクトリを中身ごと削除 |
| `cp("src", "dst")` | コピー |
| `mv("src", "dst")` | 移動・名前変更 |
| `touch("file")` | 空ファイル作成 / タイムスタンプ更新 |
| `chmod("file", mode)` | 権限（パーミッション）の変更 |

### 3. パス操作

| コマンド | 説明 |
| :--- | :--- |
| `joinpath("a", "b")` | OSに適した形式でパスを結合 |
| `abspath("file")` | 絶対パスを取得 |
| `isfile("path")` | ファイルが存在するか確認（Bool） |
| `isdir("path")` | ディレクトリが存在するか確認（Bool） |
| `splitdir("path")` | ディレクトリ名とファイル名に分割 |

### 4. ファイルの読み書き (I/O)

| コマンド                      | 説明                                   |
| :------------------------ | :----------------------------------- |
| `open("file", "r")`       | ファイルを**読み込みモード**で開く                  |
| `open("file", "w")`       | ファイルを**書き込みモード（新規作成）**で開く            |
| `read("file", String)`    | ファイル全体を**文字列として一括で読み込む**             |
| `write(io, "text")`       | 開いたファイル（io）に文字列を書き込む                 |
| `close(io)`               | ファイルを閉じる（手動でopenした場合に必要）             |
| `open(...) do io ... end` | **推奨される書き方**。ブロック終了時に**自動でclose**される |

### 5. コード例
- テキストの書き込み
```julia
open("test.txt", "w") do io
    write(io, "Hello Julia!")
end
```

- テキストの読み込み
```julia
# 全体を読み込む場合
content = read("test.txt", String)

# 1行ずつ処理する場合
open("test.txt", "r") do io
    for line in eachline(io)
        println(line)
    end
end
```

### 6. シェルコマンドの直接実行
バックティック（ \` ）を使用します。
```julia
run(`ls -l`)      # 一覧表示
run(`mkdir temp`)  # 外部コマンドでディレクトリ作成
```
