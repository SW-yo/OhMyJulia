### 1. CairoMakie
-　基本的な使い方
```julia
fig = Figure(size=(400, 400))    # size指定は無くても良い 
axis = Axis(fig[1,1],
	title="TITLE",    # グラフのタイトル
	xlabel="X",    # x軸のラベル
    ylabel="Y",    # y軸のラベル
    xticks=-10.0:5.0:10.0,    # 始点:刻み幅:終点
    yticks=-10.0:5.0:10.0,    # 始点:刻み幅:終点
    limits=((-10.0, 10.0), (-10.0, 10.0)),    # 軸の始点と終点
    aspect = AxisAspect(1),    # 軸の描画領域を正方形に固定（データの範囲に関わらず）
    # `DataAspect()`：データ座標で縦横比 1:1（データ範囲が同じ場合のみ正方形になる）
    
    )
lines!(axis, 0..5, sin)    # 線描画
scatter!(ax, X, Y)    # 散布図
contour!(ax, X, Y, Z, levels=[lebel])    # 等高線, X,Yはレンジ, Zが高さ, Levelsが等高線を描画する高さのVector
save("output.png", fig)    # 保存
fig　　　　# 表示
```
- `empty!(fig)` : 描画領域設定のクリア
- `empty!(ax)` : 描画内容のクリア
- `Point2f` : `Vector{Vector{Float64}}`を2軸のデータに変換する。
```julia
# Vector{Vector{Float64}}
x_new = [[x1, y1], [x2, y2], [x3, y3]]

# x_new:: Vector{Vector{Float64}}
scatter!(ax, Point2f.(x_new))
```
- `Vector{Vector{Float64}}`のままでは受け取れない。

### 2. 凡例とコメント
- 凡例（Legend）：

```julia
# 軸の内側に配置
axislegend(ax1, position=:rt)   # :rt=右上, :lt=左上, :rb=右下, :lb=左下

# 軸の外側（図の別セルに配置）
Legend(fig[1, 3], ax1, "Classes")
```
- テキスト注釈（text!）：

```julia
# データ座標で配置
text!(ax1, 3.0, 5.0,    # 数値は各軸の内容の単位になる
    align=(:left, :bottom),   # 水平:left/center/right, 垂直:top/center/bottom
    fontsize=12,
    text="annotation")

# 複数行
text!(ax1, 0.0, 0.0, text="line1\nline2")
```

- 図全体へのラベル（Label）：

```julia
# 軸と同じグリッド位置に Label を重ねて配置
lbl = Label(f[1, 1], "右上のラベル", 
    tellwidth = false, tellheight = false, # 軸のサイズを破壊しない設定
    halign = :right, valign = :top,       # 右上に配置
    padding = (6, 6, 6, 6),               # 内側の余白
    backgroundcolor = :yellow,            # 背景色
    strokecolor = :black,                 # 枠線の色
    strokewidth = 1                       # 枠線の太さ
)
```

※ `fig[0, :]` は図の上部（行0）、`fig[2, :]` は下部に配置

### 3. `lines`の属性
- `linestyle`
	- - `:solid`：実線（`nothing` と同等）
	- `:dot`：点線
	- `:dash`：破線
	- `:dashdot`：一点鎖線
	- `:dashdotdot`：二点鎖線
- `color`
	- - `:red` / `"red"` ：赤
	- `:blue` / `"blue"` ：青
	- `:green` / `"green"` ：緑
	- `:black` / `"black"` ：黒

### 4. グリッドの張り方
- 2次元の場合
     - 単純に（推奨） :
	    ```julia
	    # グリッドポイントの作成
		    x1_range = range(-10, 10, length=100)
		    x2_range = range(-3, 3, length=100)
		    
		# 各グリッドポイントに対してガウス核関数を適用
	    z = [f(α, μ, Σ, [x1, x2]) for x1 in x1_range, x2 in x2_range]
	    ```
	- **`collect`**  : より標準的で、メモリ効率も良い
	    ```julia
	    # グリッドポイントの作成
	    x1_range = range(-10, 10, length=100)
	    x2_range = range(-3, 3, length=100)
	    
	    # メッシュグリッド的に全ての組み合わせを作成
	    grid = collect.(Iterators.product(x1_range, x2_range))
	    
	    # 各グリッドポイントに対してガウス核関数を適用
	    z = [f(α, μ, Σ, grid[i,j]) for i in eachindex(grid, 1), j in eachindex(grid, 2)]

	    ```
    - **`SVector`** — `StaticArrays.jl`パッケージが必要で、高速だが依存関係が増える
    ```julia
	# グリッドポイントの作成
	x1_range = range(-10, 10, length=100)
	x2_range = range(-3, 3, length=100)
	
	# メッシュグリッド的に全ての組み合わせを作成
	grid = [SVector(x1, x2) for x1 in x1_range, x2 in x2_range]
	
	# 各グリッドポイントに対してガウス核関数を適用
	z = [f(α, μ, Σ, grid[i,j]) for i in 1:length(x1_range), j in 1:length(x2_range)]
	```
- 実際には、内包表記で直接 `[x1, x2]` という配列を作るのが最もシンプル。`SVector` は特に必要がなければ使わなくて大丈夫。
