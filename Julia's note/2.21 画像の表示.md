### 1. mosaicview
- 複数の画像を並べて表示
```julia
mosaic = mosaicview([img1, img2, …])

```
- 横並びにしたい時は`nrow=1`
- `fillvalue`は0.0（黒）~1.0（白）
- 画像の配列を扱う時はスプラット演算子（`...`）を使うと要素を個別の引数に展開してくれる。
```julia
mosaicview(image_array...)
```

| キーワード       | 意味                  |
| ----------- | ------------------- |
| `npad`      | 画像間の隙間（ピクセル数）       |
| `ncol`      | 列数を指定               |
| `nrow`      | 行数を指定               |
| `fillvalue` | 隙間を埋める値（デフォルト0 = 黒） |
