### ### 1. VS CodeにJulia拡張機能を入れる
1. VS Codeを開き、左側のアイコンバーにある「拡張機能（Extensions）」（四角い積み木のようなアイコン）をクリック
2. . 検索窓に **`julia`** と入力
3. 一番上に出てくるJulia公式の拡張機能（製作者が julialang のもの）を「インストール」
![[Pasted image 20260323083614.png]]

### 2. JuliaとVS Codeを紐付ける（自動）
通常はインストールした `juliaup` があれば、VS Codeが勝手にJulia本体を見つけてくれる。
確認方法
1. VS Codeで **`Ctrl + Shift + P`**（コマンドパレット）を押す。    
2. **`Julia: Start REPL`** と入力して Enter を押す。    
3. 画面の下半分に、先ほどターミナルで見た `julia>` の画面が出てくれば成功！

### 3. おすすめの初期設定（フォントと数式）
量子計算や機械学習のコードでは、$\alpha$ や $\beta$、あるいは $\ket{0}$ といった記号を使う場合、Juliaはこれらを「LaTeX形式」で入力して変換できる。
- **入力例：** `\alpha` と打って `Tab` キーを押すと $\alpha$ に変わる。    
- **フォント：** もし記号が文字化けする場合は、VS Codeの設定でフォントを `Consolas` や `Cascadia Code`、あるいは `JuliaMono`（Julia専用フォント）に設定する。

### 5. その他
- JuliaMono：フォント（[https://juliamono.netlify.app/](https://github.com/cormullion/juliamono/releases/download/v0.062/JuliaMono-ttf.zip)）
	1. フォント（.zip）をダウンロード
	2. 解凍して右クリックでインストール（まずはRRegularだけでOK）
	3. VSCodeで`ctrl`+`,`
	4. `font family`と入力
	5. `Editor: Font Family` の入力欄の一番先頭に`'JuliaMono',` と追記
	6. （オプション）`Font Ligatures` を検索し、「Edit in settings.json」 をクリックして `"editor.fontLigatures": true` に書き換えると、`->` が矢印 `→` につながって見えるようになる。
- 文字サイズの調整
		1. `Ctrl` + `,`** (カンマ) を押して設定を開く。
		2. 上部の検索バーに `terminal font size` と入力する。
		3. - 同じ設定画面で `terminal font family` と検索する。
		4. `Terminal › Integrated: Font Family` の欄に `JuliaMono` と入力する。
