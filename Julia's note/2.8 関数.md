### 2. 関数の定義
- `function[関数名]([引数])`の後にインデント
- インデント内で`return [戻り値]` 
- 最後に`function`と同じレベルで`end`
例：
```
function f(x)
    y = 1 / x + 3 * x
    return y
end
```
- 直接retuenに内容を書いてもOK
- 数学同様`*`も省略できる
例：
```
function f(x)
    return 1 / x + 3x
end
```
- 宣言しなくてもOK
例：
```
f(x) = 1 / x + 3x
```
- 引数が複数でもOK
例：
```
function say(str, num)
    for i in 1:num
        println(str)
    end
end

say("あら", 5)
```
- 戻り値が複数でもOK
- 複数の戻り値を1行で受け取ることもできる。
例：
```
function  split(vec, k)
    v1 = vec[1:k]
    v2 = vec[k+1:end]
    return v1, v2
end

x = [1; 2; 3; 4; 5]
x1, x2 = split(x, 2)
println(x1)
println(x2)
```

### 3. !マーク
- 関数名の最後に`!`を付けた場合、その関数は「破壊的」
- 引数を変更する場合に付ける（慣習）。
例：
```
arr = [10; 20; 30]

# 強制的に一番最後へ要素を追加
push!(arr, 40)

# 強制的に一番最後の要素を削除
pop!(arr)
```

### 4. 関数の引数

- 一般的な引数

| 用途・イメージ                     | 代表的な引数名                              | 例（関数シグネチャ）                              |
|------------------------------------|--------------------------------------------|---------------------------------------------------|
| 一次元ベクトル（数値の列）         | v、x、vec、data、values                   | `function foo(v::Vector{Float64}) … end`        |
| 二次元行列（線形代数の対象）       | A、M、X、Y、mat                           | `function bar(A::Matrix{Float64}) … end`        |
| 任意次元の配列（汎用的に）         | arr、a、tensor、grid                      | `function baz(arr::Array{T}) … end`             |
| インデックスや座標系（配列の要素を指す） | idx、i、j（ただしループ変数）            | `function get_val(arr::Array, idx::Int) … end` |
| データセット全体（複数の配列をまとめて渡す） | data、dataset、samples                 | `function train(data::Vector{Vector{Float64}}) … end` |
