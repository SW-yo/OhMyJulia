### 2. 関数の定義
- `function[関数名]([引数])`の後にインデント
- インデント内で`return [戻り値]` 
- 最後に`function`と同じレベルで`end`
例：
```
function f(x)
    y = 1 / x + 3 * x
    return y
end
```
- 直接retuenに内容を書いてもOK
- 数学同様`*`も省略できる
例：
```
function f(x)
    return 1 / x + 3x
end
```
- 宣言しなくてもOK
例：
```
f(x) = 1 / x + 3x
```
- 引数が複数でもOK
例：
```
function say(str, num)
    for i in 1:num
        println(str)
    end
end

say("あら", 5)
```
- 戻り値が複数でもOK
- 複数の戻り値を1行で受け取ることもできる。
例：
```
function  split(vec, k)
    v1 = vec[1:k]
    v2 = vec[k+1:end]
    return v1, v2
end

x = [1; 2; 3; 4; 5]
x1, x2 = split(x, 2)
println(x1)
println(x2)
```
