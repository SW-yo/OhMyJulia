1. 魔法のドット `.` (ブロードキャスト)
	- 関数の後ろに `.` をつけるだけで「配列の全要素に対して実行せよ」という命令になる。
例：
```
x = [1, 2, 3]

# for文を使わずに、全要素を2倍にする
y = x .* 2  

# 全要素に sin を計算する
z = sin.(x)
```
- 「基礎」として重要性
	- **直感的:** 行列 `A` の全要素に `10` を足したいとき、`A .+ 10` と書くだけで済む。
	- **爆速:** Julia の内部で最適化されるため、自分で `for` 文を書くよりも速く、メモリ消費も抑えられる。
	- **GPU との相性:** このドット演算子はそのままGPUへの命令に変換できる。
例：
```
A = [1 2; 3 4]

# パターン1 (行列の掛け算)
B1 = A * A

# パターン2 (要素ごとの掛け算)
B2 = A .* A
```
