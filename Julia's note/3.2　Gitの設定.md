### 1. Git
- 該当のディレクトリに`cd`
- そこで`git init`
- `git config user.name "(name)"`
- `git config user.email "(email)"`
- `git branch -m master main`
### 2. GitHub
- `git remote add origin https://github.com/(GitHub URL)`
- `git remote -v`
