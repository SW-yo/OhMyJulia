### 2. uv環境作成
- Python用のuv環境はJuliaと分けた方がいい（.tomlが混乱する）。
- Python用のディレクトリに移動（`cd`）
- 移動したディレクトリでuvを初期化（`uv init`）
- 仮想環境（`.venv`）を作成（`uv venv`）
- 必要なライブラリをインストール（例：`uv add numpy`）
- 既に組み込まれている標準ライブラリ（time、math、os、sysなど）をインストールしようとするとエラーになる。
- 複数のライブラリをインストールする場合は半角空白区切り（`uv add numpy pandas matplotlib`）
- インポートされているライブラリの確認は`uv tree`（`uv pip list`

### 3. Pythonの起動
- **`Ctrl` + `Shift` + `P`** を押す。
- **`Python: Select Interpreter`** と入力
- リストの中から **`./.venv/Scripts/python.exe`** （「Recommended」ではなく「venv」と書いてあるもの）を選択
- **ここでいったんルートディレクトリに戻る。ファイル操作するにはプロジェクトのディレクトリに移動**
- uvは自動的にさいてきなPythonwoダウンロードしてきてくれる（？）
- Pythonのバージョンの確認は`uv python list
- 特定のバージョンを使いたいときは`uv venv --python 3.xx`で指定

## 4. Pythonの実行
- 右上の「▷（実行ボタン）」をクリック
- ターミナルで手動実行はプロジェクトのディレクトリに移動してから
```
uv run [filename].py
```
