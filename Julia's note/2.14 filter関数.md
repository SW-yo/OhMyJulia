### 1. filter()
- データフレーム（配列（ベクター）や辞書などのコレクション）から特定の条件を満たす要素のみを抽出するための関数
- https://docs.julialang.org/en/v1/base/collections/#Base.filter
```
# 偶数のみを抽出
arr = [1, 2, 3, 4, 5, 6]
filtered_arr = filter(x -> iseven(x), arr)
# 結果: [2, 4, 6]
```

### 2. 使い方
- `filter(source => f::Function, df)`
- `source` : フィルタリング対象の列
- `f::Function` : 無名関数で記載するフィルタ条件
- `df` : フィルタリングするデータフレーム
