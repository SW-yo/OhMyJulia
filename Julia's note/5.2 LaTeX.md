### 1. 基本操作
- コード内でLaTeXが使える。
- `\`+LaTeX記号入力したらtabキーで変換。

### 2. 平方根
- `\sqrt`+tab
- 平方根の対象を`()`で囲む。
- `sqrt()`関数でも可。

### 3. 指数関数
- `\euler`+tab+`^`
- 指数は`()`で囲む。
- `exp()`関数でもの方が良い。
### 4. 分数
- `\frac{numerator}{denominator}`

### 5. 総和・総積・積分

- 総和：`\sum`
- 総積：`\prod`
- 積分：`\int`
- 添字
	1. 添字が横：`\sum_{k=1}^n`
	2. 添字が上下：`\sum\limits_{k=1}^n`

### 6. 波括弧
- 下向き波括弧 : `\underbrace{a + b + c}_{注釈}`

### 7. 添字
- 文字列の下付き : `\underset{下の文字}{上の文字}`

### 8. 空白

| コマンド                     | 空白の幅   | 特徴・用途                     |
| :----------------------- | :----- | :------------------------ |
| `\,`                     | 極小     | 数式内の微調整用（ab の間など）         |
| `\ ` （バックスラッシュ + 半角スペース） | 通常     | 通常の半角スペースと同じ幅             |
| `\quad`                  | 全角1文字分 | フォントサイズ1文字分の幅             |
| `\qquad`                 | 全角2文字分 | `\quad` の2倍の幅（数式を並べるときなど） |
### 9. 等号
- 基本のイコール・不等号・否定

| 記号 | コマンド | 意味 |
| :---: | :--- | :--- |
| = | `=` | 等しい（イコール） |
| ≠ | `\neq` または `\ne` | 等しくない（ノットイコール） |
| < | `<` | 小なり |
| > | `>` | 大なり |
| ≤ | `\leq` または `\le` | 以下（小なりイコール） |
| ≥ | `\geq` または `\ge` | 以上（大なりイコール） |
| ≡ | `\equiv` | 合同、恒等 |

- ニアリーイコール・近似・相似

| 記号 | コマンド | 意味 |
| :---: | :--- | :--- |
| ≈ | `\approx` | ほぼ等しい（近似・ニアイコール） |
| ≒ | `\fallingdotseq` | ほぼ等しい（主に日本で使われる「≒」） |
| ≃ | `\simeq` | ほぼ等しい（漸近的に等しい） |
| ~ | `\sim` | 相似、比例する、〜のオーダー |
### 10. 行列
| **LaTeXコマンド**                                  | **記述結果**                                       |
| ---------------------------------------------- | ---------------------------------------------- |
| `\begin{matrix} a & b \\ c & d \end{matrix}`   | $\begin{matrix} a & b \\ c & d \end{matrix}$   |
| `\begin{pmatrix} a & b \\ c & d \end{pmatrix}` | $\begin{pmatrix} a & b \\ c & d \end{pmatrix}$ |
| `\begin{bmatrix} a & b \\ c & d \end{bmatrix}` | $\begin{bmatrix} a & b \\ c & d \end{bmatrix}$ |
| `\begin{vmatrix} a & b \\ c & d \end{vmatrix}` | $\begin{vmatrix} a & b \\ c & d \end{vmatrix}$ |
| `\begin{Vmatrix} a & b \\ c & d \end{Vmatrix}` | $\begin{Vmatrix} a & b \\ c & d \end{Vmatrix}$ |
