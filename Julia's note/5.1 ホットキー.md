### 1. ホットキーにする記号
- Microsoft PowerToysの"Keyboard Manager"をインストール
	![[Pasted image 20260324145709.png]]
- 入れにくい記号
	- 〉（ケット）⇒F12
	- 〈（ブラ）⇒F11
	- ⊗（テンソル積）⇒F1
	- ⊕（直和/排他的論理和XOR）⇒F2
	- ≈（近似）⇒F3

### 2. 変換で入れる記号
- ∈（含む）：ふくむ
- ∴（ゆえに）：ゆえに
- ∵（なぜならば）：なぜならば

### 3. Juliaの入力
- LaTeXが使える。`\`で始まるLaTeXを入れてtabキーで変換できる。
	- ≈：`\approx`
	- ⊗：`\otimes`
	- 〉：`\rangle`
	- 〈：`\langle`
- Julia REPL, Jupyter Notebook, VS Code (Julia拡張機能) で動作する。
