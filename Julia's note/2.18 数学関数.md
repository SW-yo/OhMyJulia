### 1. 基本的な算術・指数・対数
- 数値 $x$ に対して直接使える。
- 行列に適用したい場合は「ドット」をつけて `exp.(A)` のように書くことができる。

|     関数     |    数学的な意味     | 備考                                       |
| :--------: | :-----------: | :--------------------------------------- |
| `sqrt(x)`  |  $\sqrt{x}$   | 平方根。負の数を入れるとエラー（複素数は `sqrt(complex(x))`） |
| `cbrt(x)`  | $\sqrt[3]{x}$ | 立方根                                      |
|  `exp(x)`  |     $e^x$     | 指数関数                                     |
|  `ln(x)`   |    $\ln x$    | 自然対数（底が $e$）                             |
| `log10(x)` | $\log_{10} x$ | 常用対数（底が $10$）                            |
|  `abs(x)`  |     \|x\|     | 絶対値                                      |

---
### 2. 三角関数
- 三角関数は**ラジアン単位**と**度（degree）単位**の両方が用意されている。
	* **ラジアン単位:** `sin(x)`, `cos(x)`, `tan(x)`, `asin(x)` (arc-sin)
	* **度単位（末尾にd）:** `sind(x)`, `cosd(x)`, `tand(x)`
	    * 例：`sind(90)` は `1.0` を返します。`sin(π/2)` と書く手間が省ける。
	* **$\pi$ を掛けた計算:** `sinpi(x)`
	    * `sin(π * x)` を計算する。
	    * `sinpi(1)` は正確に `0.0` を返すため精度が必要な時に重宝する。

---
### 3. 丸め処理（近似値）
- 小数点の丸め処理
	- **`round(x, digits=i)`** :  xが対象、iが桁数（`digits=`が必要）
- 整数に丸める（数値を整数に丸める関数も選べる）。
	* **`round(x)`** : 四捨五入（正確には、もっとも近い整数へ。中間値は偶数側へ）
	* **`floor(x)`** : 切り捨て（$-\infty$ 方向へ）
	* **`ceil(x)`** : 切り上げ（$+\infty$ 方向へ）
	* **`trunc(x)`** : $0$ 方向への切り捨て（正の数は `floor`、負の数は `ceil` と同じ挙動）

---
### 4. 行列・ベクトル統計
- 配列全体を対象にする関数
	* **`sum(A)`**: 合計
	* **`prod(A)`**: 全要素の積
	* **`maximum(A)` / `minimum(A)`**: 最大値 / 最小値
	* **`argmax(A)/argmin(A)`** : 最大値/最小値のインデックス
	* **`findmax(A)/findmin(A)`** : 最大値/最小値の値とインデックスのタプル
- `using Statistics` が必要
	* **`mean(A)`**: 平均
		* `Vector{Vector{Float64}}`に対しては`{}`の内側の`Vector`の平均ベクトルが返る(\[μ1, μ2, ..., μn\])
		* `Matrix`に対しては
		```julia
		mean(X)          # → スカラー（全要素の平均）✗
		mean(X, dims=1)  # → 1×4 の行列（列ごとの平均）✓
		```
	* **`var(A)`** : 不偏分散（n-1 で割った値） 
	- **`var(A, corrected=false)`** : 標本分散（n で割った値） 
	- **`std(A)`** : 不偏標準偏差 
	- **`std(A, corrected=false)`** : 標本標準偏差
	- **`cov(A)`** : 共分散行列

### 5. 微分・積分

|             関数             |             数学的な意味              | 備考                                             |
| :------------------------: | :-----------------------------: | :--------------------------------------------- |
|    `sum(f(i), i, a, b)`    |      $\sum_{i=a}^{b} f(i)$      | 総和（LaTeXコード: `\sum_{i=a}^{b} f(i)`）            |
|   `prod(f(i), i, a, b)`    |     $\prod_{i=a}^{b} f(i)$      | 総積（LaTeXコード: `\prod_{i=a}^{b} f(i)`）           |
| `integrate(f(x), x, a, b)` |     $\int_{a}^{b} f(x) dx$      | 定積分（LaTeXコード: `\int_{a}^{b} f(x) dx`）          |
|        `diff(y, x)`        |         $\frac{dy}{dx}$         | 常微分（LaTeXコード: `\frac{dy}{dx}`）                 |
|      `diff(y, x, n)`       |     $\frac{d^{n}y}{dx^{n}}$     | 高階常微分（LaTeXコード: `\frac{d^n y}{dx^n}`）          |
|       `pdiff(f, x)`        | $\frac{\partial f}{\partial x}$ | 偏微分（LaTeXコード: `\frac{\partial f}{\partial x}`） |
|    `limit(f(x), x, a)`     |      $\lim_{x \to a} f(x)$      | 極限（LaTeXコード: `\lim_{x \to a} f(x)`）            |

### 💡 Juliaらしい便利な書き方
Juliaでは、数学の数式に近い見た目で書ける工夫がある。
* **係数の省略:** `2π` や `3x` のように、数字と変数の間の `*` を省略できる。
* **比較の連結:** `0 < x < 10` のような数学的な不等式の書き方がそのまま通る。

>Tips： 関数の使い方がわからなくなったら
- コンソール（REPL）で `?` を入力してから関数名を打つと詳細な説明と使い方が英語で表示される。ただしドキュメントのママなので長い。
- `methods([関数名])`： 関数で引数だけ確認する。
- `apropos("[keyword]")`：キーワードで関連関数を探す。ただしキーワードは英語で。
