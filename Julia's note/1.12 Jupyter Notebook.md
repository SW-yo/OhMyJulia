### 1. IJliaのインストール
```
using Pkg
Pkg.add("IJulia")
```

### 2. VSCodeでJupyter Notebookを使う場合のJuliaの扱い
- ターミナルでREPLを起動したり、環境をactivateする必要は基本的にない。
- activateはJupyter Notebookのセルで実行可能。
```julia
using Pkg
Pkg.activate(".")   # カレントディレクトリのProject.tomlを使う
# or
Pkg.activate(@__DIR__)
```
- ターミナルのREPLが必要になるケース
	- パッケージの追加・削除（`add`, `rm`など）をPkg modeでやりたいとき
	- Julia拡張機能のLanguage Server（補完・エラー表示）の設定を変えたいとき
	- ノートブック外でスクリプトを実行したいとき
