### 1. Juliaup
- 基本的にはMicrosoft Storeから「Julia」でOK。
- ただしそのままだとエラーが返ってきた。「本体（Juliaup）自体はインストールされているものの、どのバージョンのJuliaを使うかという設定（チャネル）が空っぽの状態」らしい。
- 解決策は`juliaup add release`
- インストール結果の確認は`julia --version`
- `julia`でJuliaの対話画面（REPL）が起動するようになる。

### 2. Juliaの終了
- `exit()`
- `ctrl`+`D`
### 3. GPUをJuliaに紐づけ
1. **パッケージモードへ切り替え** `]` キーを押す（プロンプトが `(@v1.12) pkg>` に変わる）。
2. **CUDAパッケージを追加**
	```
	add CUDA
	```
     ※ここでGPU用のドライバーやツールキットが自動ダウンロードされる。数分かかることがある。
3. **動作確認** `Backspace` キーで元の `julia>` に戻り、以下を入力
    ```
    using CUDA
    CUDA.functional()
    ```
	これが **`true`** と返ってくればOK。
